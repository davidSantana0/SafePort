Quem somos

O SafePort é um sistema de cadastro criado com foco na segurança e na simplicidade. Este projeto foi desenvolvido com o objetivo de oferecer um ambiente seguro para armazenar informações pessoais, de forma didática e acessível, especialmente para usuários a partir de 10 anos.

Com uma interface amigável e validações pensadas para proteger seus dados, o SafePort pode ser utilizado em portais, sites educacionais ou até mesmo pequenos negócios que precisam de uma solução prática e segura.

Além disso, o sistema valoriza a acessibilidade, garantindo que qualquer pessoa consiga navegar e utilizar seus recursos com facilidade.

Tecnologias utilizada

  • HTML
  • CSS
  • JavaScript
  • node
  • Express

Um pouco da nossa história

Tudo começou com uma ideia simples: criar um sistema de cadastro que fosse mais do que só campos e botões. A gente queria segurança de verdade, praticidade e uma interface que qualquer pessoa pudesse usar sem medo.

No começo, éramos só um dev e uma designer curiosos, quebrando a cabeça com o design, tentando entender como o back-end funcionava e tropeçando em cada linha de código.

Com o tempo, o SafePort foi ganhando forma. Cada erro corrigido, cada função aprendida, cada madrugada de código… tudo isso virou parte da história do projeto.

Hoje, o SafePort é resultado de muita dedicação, paciência, noites viradas e uma boa dose de persistência. Mais do que um sistema, ele é a prova de que quando a gente se junta com um propósito — mesmo que pequeno — dá pra construir algo útil, bonito e seguro.

Por que o SafePort é diferente

Porque aqui segurança não é só palavra bonita no código — é compromisso real. A gente junta simplicidade, praticidade e proteção forte pra você guardar seus dados sem dor de cabeça.

O que torna nosso sistema único

É a combinação de design acessível, código pensado pra todo mundo usar, e uma blindagem na proteção dos seus dados que faz a diferença. SafePort é feito pra ser seguro, fácil e confiável, sem enrolação.
